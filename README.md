# 常用快捷键
清理屏幕
- ctrl+L

停止终端命令/终止当前进程
- ctrl+C

挂起当前进程（发送 SIGTSTP 信号），进程被暂停并放入后台。你可以用 fg 恢复到前台，或用 bg 在后台继续运行。
- Ctrl + Z
强制终止当前进程（发送 SIGQUIT 信号），比 Ctrl + C 更强硬，通常会生成核心转储（core dump）。
- Ctrl + \
